Bienvenido a su herramienta para la gestión de personas.


ESta herramienta esta diseñada para que pueda trabajar con la base de datos de personas. En ella encontrará las siguientes opciones:

  • Agregar una nueva persona a la base de datos existente.
  • Rellenar el formato de adopción para la persona que desee.
  • Consultar la tabla con la información de todas las personas.
